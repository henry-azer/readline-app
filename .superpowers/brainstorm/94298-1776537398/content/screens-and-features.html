Screens & Features Breakdown

Every screen from the designs mapped to implementation

Onboarding Flow (3 mandatory steps)

Step 1: Welcome
"Read at your pace. Grow at your speed."

• Streak counter animation (12 Days)
• "A Mindful Journey" tagline
• App value props: Distraction Free, Intellectual Depth
• "GET STARTED" CTA button
• "Join 20k+ readers" social proof
Step 2: Assessment
"How fast do you read?"

• 4 selectable level cards:
  → Beginner (50-150 WPM)
  → Intermediate (150-250 WPM)
  → Advanced (250-350 WPM)
  → Expert (350-500 WPM)
• Each with icon, description
• "CONTINUE JOURNEY" CTA
Step 3: First Import
"Import your first PDF"

• Step indicators (3 dots)
• Drop zone with upload icon
• "Supports PDF, EPUB, and Text"
• "CHOOSE FROM FILES" primary CTA
• "TRY WITH SAMPLE TEXT" secondary
• Saves onboarding_complete flag

Main App (4 Bottom Tabs + 2 Full-Screen)

Tab 1: Home
Empty State:
• Stats bar (Documents, Speed, Words)
• Book stack image placeholder
• "No PDF loaded" message
• "IMPORT PDF" CTA button
• Reading Tip card

Active State:
• Current document info
• Continue reading CTA
• Recent activity summary
• Quick stats overview
Tab 2: Library
"My Library"
• Document count ("12 Active Documents")
• Grid/List toggle (top right)
• Filter tabs: All / Reading / Completed
• PDF cards with:
  → Cover thumbnail/preview
  → Filename
  → Page progress (142/305)
  → Progress bar (colored)
  → Completion badge
• Floating "+ IMPORT PDF" button
Tab 3: Vocabulary
"Personal Word Bank"
• Search bar + settings gear
• Filter chips: All / New / Learning / Mastered
• Daily Insight card (tip/recommendation)
• Word cards with:
  → Source PDF tag (colored)
  → Word in large serif font
  → Context quote (italic)
  → Audio pronunciation button
  → Added date + source link
  → Swipe to mark learned
• "Daily Review Bloom" session card
• Manual add (+) FAB
Tab 4: Analytics
Light theme version:
• Gradient streak card (7 days + dots)
• 6 stat cards (2x3 grid):
  → Total Sessions, Words Read
  → Avg Speed, Total Time
  → Curr Streak, Longest Streak
• Growth Insight banner
• Reading Volume bar chart (7 days)
• Recent Activity list

Dark theme version:
• Fire icon streak with "Unstoppable"
• Weekly activity dots (Mon-Sun)
• "Archivist's Note" (personalized quote)
• Current Goal display
• Reading Velocity line chart (Week/Month)
• Curated Progress (per-book cards)
Full-Screen: Active Reading
Three-zone text display:
• Top: Small italic text (already read)
• Center: Large bold text (current focus)
• Bottom: Regular text (upcoming)

Header:
• Back arrow + "Read-It" logo
• Streak badge ("14 Day Streak")
• More options menu

Controls bar (bottom):
• Typography toggle (Tt)
• Speed display (WPM)
• Play/Pause button (center)
• Max speed indicator
• Save/bookmark button

Vocab highlight:
• Tappable words with highlight
• "Current Context" bar shows selected word
Full-Screen: Settings
Reading Settings:
• Speed (WPM) slider → 350
• Line Spacing slider → 1.5x
• Font Size slider → 18pt
• Focus Window slider → 3 lines

Appearance:
• Theme Mode: Light / Dark / System
• Font Family: Inter / Literata / Newsreader

Advanced:
• Vocab Collection toggle
• Reading Analytics toggle
• "RESET TO DEFAULTS" button

Live Preview:
• Real-time text preview with current settings
• Shows focus mode, font, spacing applied
