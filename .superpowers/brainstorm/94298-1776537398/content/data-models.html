Core Services

PdfProcessingService
• Extract text via Syncfusion
• Calculate word count
• Score complexity (Beginner→Expert)
• Split into page chunks
• Estimate reading time per level
• Validate PDF before import
• Detect complex vocabulary words
ReadingEngineService
• WPM-based auto-scroll timing
• Three-zone position tracking
  (past / focus / upcoming)
• Play / pause / resume / stop
• Speed adjustment mid-session
• Focus score calculation
• BehaviorSubject for position state
• Proper stream lifecycle (no close-on-stop bug)
VocabularyService
• Auto-detect uncommon words
• Manual word save from reading
• Spaced repetition scheduler
  (nextReviewAt calculation)
• Mastery progression logic
  (new → learning → mastered)
• Daily review session generator
• Word stats aggregation
StreakService
• Track daily reading activity
• Calculate current streak
• Update longest streak
• Weekly activity array (Mon-Sun)
• Milestone detection
  (7d, 14d, 30d, 100d labels)
• Reset on missed day
• BehaviorSubject for streak state
