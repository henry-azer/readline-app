Core Patterns (Identical to Padlock)

State Management
RxDart BehaviorSubject
ViewModels expose streams
UI uses StreamBuilder
Explicit dispose() lifecycle
Dependency Injection
GetIt service locator
Single injection.dart file
LazySingleton for services
Singleton for config
Navigation
GoRouter
StatefulShellRoute for tabs
4 branches: Home, Library, Vocab, Analytics
Reading + Settings outside shell
Storage
Hive
Documents, sessions, preferences
Vocabulary words with mastery
Streak tracking

Data Flow

Architecture layers
Presentation — Screens → StreamBuilder → ViewModels (BehaviorSubject)
↕
Core Services — PDF Processing, Reading Engine, Vocabulary, Streak
↕
Data — Contracts → Repositories → Hive DataSources
↕
Storage — Hive Boxes (documents, sessions, preferences, vocabulary, streaks)
