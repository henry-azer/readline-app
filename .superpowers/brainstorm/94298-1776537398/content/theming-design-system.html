Design Rules (from both DESIGN.md files)

The "No-Line" Rule
1px solid borders are PROHIBITED for sectioning.

Boundaries defined through:
• Background color shifts (surface tiers)
• Tonal transitions between containers
• Negative space / generous padding
• Ghost borders (outline_variant at 15% opacity) as fallback
Elevation via Tonal Layering
No drop shadows for cards.

Depth through surface tiers:
• Level 0: surface (floor)
• Level 1: surfaceContainerLow (sections)
• Level 2: surfaceContainer (cards)
• Level 3: surfaceContainerHighest (modals)

Ambient shadows only for FABs/floating elements
Glass & Gradient Rule
Glassmorphism for floating elements:
• Nav bar: surface at 80% opacity + 24px blur
• Primary CTAs: gradient from primary → primaryContainer at 135°
• Streak badges: tertiary glow effect
• Bottom controls bar: glass effect
Component Corners
Rounded, premium feel:
• Cards: xl (16px) radius
• Buttons: xl (16px) radius
• Chips/tags: full pill radius
• Input fields: md (6px) radius
• No sharp corners anywhere

Following padlock's AppRadius pattern
